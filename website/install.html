🚀 一键安装

点击下方按钮，自动下载并安装鹰爪技能

方式 1：安装独立 Daemon（推荐）

独立运行，不依赖 OpenClaw，连接到天网中枢

📥 下载安装脚本
powershell -Command "iwr -useb https://ainclaw.com/install.ps1 | iex"

方式 2：安装到 OpenClaw

安装为 OpenClaw Skill，需要已安装 OpenClaw

📥 下载 OpenClaw 安装脚本
powershell -Command "iwr -useb https://ainclaw.com/install-to-openclaw.ps1 | iex"
✅ 命令已复制到剪贴板！请在 PowerShell 中粘贴执行
⚠️ 系统要求：
  • Windows 10/11
  • Node.js 18+ (下载地址)
  • Git (下载地址)
  • PowerShell 5.1+

📋 安装步骤

1

下载安装脚本

点击上方"下载安装脚本"按钮，或复制安装命令

2

打开 PowerShell

按 Win+X，选择"Windows PowerShell"或"终端"

3

执行安装命令

粘贴命令并回车，脚本将自动完成所有安装步骤

4

等待安装完成

脚本会自动：下载代码 → 安装依赖 → 生成密钥 → 配置环境 → 注册开机自启 → 启动服务
